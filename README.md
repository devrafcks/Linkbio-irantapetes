# Iran Tapetes Orientais 

Site de página única no formato *Linktree*, desenvolvido para centralizar canais de atendimento e vendas da **Iran Tapetes Orientais**.  
Projeto orientado a performance mobile, com estética premium alinhada a luxo e exclusividade.

## Objetivo
Centralizar acessos estratégicos da marca em um único ponto, reduzindo fricção e aumentando conversão em dispositivos móveis.

## Destaques do Design
- Linguagem visual sofisticada e minimalista  
- Experiência **mobile-first**  
- Interface limpa, direta e focada em ação  
- Uso de efeitos modernos sem comprometer legibilidade ou performance

## Tecnologias Utilizadas
- **HTML5** — Estrutura semântica e acessível  
- **Tailwind CSS** — Estilização utilitária e responsividade  
- **Font Awesome** — Ícones sociais e de interface  
- **Google Fonts** — Tipografia personalizada (Berkshire Swash, Inter, Oswald)  
- **CSS Moderno** — Mesh Gradient animado, Glassmorphism e layout com CSS Grid

## Resultado
Uma página leve, elegante e funcional, preparada para uso comercial real e compartilhamento direto em redes sociais.
